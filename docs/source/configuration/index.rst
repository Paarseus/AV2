=============
Configuration
=============

Configuration and calibration guides for the AV2.

.. contents:: In This Section
   :local:
   :depth: 2

Overview
========

The AV2 uses configuration files and calibration data to customize behavior for different environments and vehicles.

Configuration Files
===================

+------------------------+---------------------------+
| File                   | Purpose                   |
+========================+===========================+
| config/dwa_config.yaml | DWA planner parameters    |
+------------------------+---------------------------+
| config/vehicle.yaml    | Vehicle specifications    |
+------------------------+---------------------------+
| config/sensors.yaml    | Sensor configurations     |
+------------------------+---------------------------+
